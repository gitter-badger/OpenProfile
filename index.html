Welcome to the OpenProfile project

Overview

What Is OpenProfile?

OpenProfile is a free and open-source easy to use autobiography and biography creator written in Python.

How Did This Become?

I wanted to learn Python after I had gotten my RaspberryPi. So I started learning it. My first Python project was I-wantz-doges, a project for developer shits and lols, it wasn't a serious project, anyway I continued developing the I-wantz-doges application until I thought came to my mind, that thought was the idea of OpenProfile.
